# Customer Income Category Prediction using Machine Learning

## Project Overview

This project focuses on predicting customer income categories using machine learning techniques based on demographic, financial, and behavioral customer data.

The project demonstrates a complete data science workflow including:

* Data preprocessing
* Missing value handling
* Exploratory Data Analysis (EDA)
* Feature preparation
* Machine learning workflow development

The objective of the project is to analyze customer-related factors and prepare datasets for predictive machine learning tasks.

---

## Objectives

* Perform data preprocessing for customer datasets
* Prepare structured datasets for machine learning workflows
* Analyze customer demographic and financial features
* Improve understanding of data cleaning and preprocessing techniques

---

## Dataset

Dataset used:

* Credit Card Customer Dataset

The dataset contains customer demographic, financial, and behavioral information for machine learning analysis and prediction tasks.
